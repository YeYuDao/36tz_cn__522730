<br/></br>### 课程介绍:
该项目是基于最新版本 React 的全栈项目，此项目为一个前后台分离的招聘的 SPA, 包括前端应用和后端应用 。包括用户注册/登陆, Boss/牛人列表, 实时聊天等功能模块 。前端技术: React全家桶 + ES6 + Webpack 等、后端技术: NodeJS + Express + MongoDB + SocketIO等 。采用模块化、组件化、工程化的模式开发。 并自定义了Redux和React-Redux库。
